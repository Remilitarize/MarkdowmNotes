## 惯性力 质点的达朗贝尔原理

人用手推车：F' = -F = -ma
力 F' 是由于小车具有惯性，力图保持原来的运动状态，对于施力物体（人手）产生的反抗力，称为小车的 **惯性力**。

**定义：F<sub>I</sub> = -ma**
**加速运动的质点，对迫使其产生加速运动的物体的惯性反抗的总和。**

惯性力的大小等于质点的质量与其加速度的乘积，方向与加速度的方向相反，**不作用于运动质点本身，而作用于周围的施力物体**

质点的达朗伯原理：非自由质点 M，质量 m，受主动力 F，约束反力 F<sub>N</sub>，合力 F<sub>R</sub> = F + F<sub>N</sub> = ma。
**F + F<sub>N</sub> + F<sub>I</sub> = 0**

## 质点系的达朗伯原理

设有一质点系由n个质点组成，对每一个质点，有 F<sub>i</sub> + F<sub>Ni</sub> + F<sub>Ii</sub> = 0（i = 1, 2, ..., n）
对整个质点系，主动力系、约束反力系、惯性力系形式上构成平衡力系。这就是 **质点系的达朗伯原理**。
**&Sigma;F<sub>i</sub> + &Sigma;F<sub>Ni</sub> + &Sigma;F<sub>Ii</sub> = 0**
**&Sigma;m<sub>o</sub>(F<sub>i</sub>) + &Sigma;m<sub>o</sub>(F<sub>Ni</sub>) + &Sigma;m<sub>o</sub>(F<sub>Ii</sub>) = 0**

F<sub>i</sub><sup>(e)</sup> 为作用于第 i 个质点上外力的合力。
F<sub>i</sub><sup>(i)</sup> 为作用于第 i 个质点上外力的合力。
**&Sigma;F<sub>i</sub><sup>(e)</sup> + &Sigma;F<sub>Ii</sub> = 0**
**&Sigma;M<sub>o</sub>(F<sub>i</sub><sup>(e)</sup>) + &Sigma;M<sub>o</sub>(F<sub>Ii</sub>) = 0**
质点系的达朗贝尔原理：作用在质点系上的外力与虚加在每个质点上的惯性力在形式上组成平衡力系。

## 刚体惯性力系的简化

刚体作平动：
刚体平动时惯性力系合成为一过质心的合惯性力。
F<sub>IR</sub> = -Ma<sub>C</sub>

定轴转动刚体：
主矢：惯性力系的主矢为主惯性力的矢量和，即 F<sub>IR</sub> = -ma<sub>C</sub>
主矩：惯性力系的主矩为诸惯性力对简化中心的力矩的代数和，即：M<sub>IO</sub> = -J<sub>o</sub>&epsilon;

特殊情况讨论：

- 刚体作匀速转动，转轴不通过质点 C，切向惯性力等于零，只有法向惯性力 F<sub>IR</sub> = Me&omega;<sup>2</sup>，主矩 M<sub>IO</sub> = 0。
- 转轴过质点 C，但 &epsilon; &ne; 0，惯性力偶 M<sub>I</sub> = -J<sub>C</sub>&epsilon;，切向惯性力的合力等于零，法向惯性力为零，F<sub>IR</sub> = 0。
- 刚体作匀速转动，且转轴过质心，则 F<sub>IR</sub> = 0，M<sub>IC</sub> = 0。

刚体作平面运动：
随基点（质点 C）的平动：F<sub>IR</sub> = -Ma<sub>C</sub>
绕通过质心轴的转动：M<sub>IC</sub> = -J<sub>C</sub>&alpha;
